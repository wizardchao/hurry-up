# 开发一个chrome浏览器插件

## 实现要求
点开时会有弹框出现，弹框里内容如下
1.会有当前时间，年月日小时-分钟-秒，星期几要求实时显示
2.下面一行会显示当前时间距离18:30还剩多少小时/分钟/秒，如超过则显示已过下班时间
3.下面一行会显示今天距离周末开始还剩多少天，如超过则显示已放假
4.下面一行会显示距离15号还剩多少天，如超过则显示月已过半
5.下面一刚会显示距离月底还剩多少天
6.下面一行会显示距离年底还剩多少天
7.下面一行会判断距离端午节开始还剩多少天，如超过则不显示
8.下面一行会判断距离中秋节开始还剩多少天，如超过则不显示
9.下面一行会判断距离国庆节开始还剩多少天，如超过则不显示

## 样式要求
外观要简洁大气，适量加入一些动态特效
